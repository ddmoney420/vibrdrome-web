# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Vibrdrome, please report it privately via email:

**vibrdrome@gmail.com**

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and provide updates as we work on a fix. Please do not open a public issue for security vulnerabilities.

## Supported Versions

Only the latest release is actively supported with security updates.
