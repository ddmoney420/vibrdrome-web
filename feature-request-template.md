# Feature Request Guidelines (Copy into Discord Post Guidelines)

Please use the following template when submitting a feature request. This helps us understand what you need and why.

## Template

```
[Title - Brief description of the feature]

**Type:** [Enhancement / Modification / Quality of Life / Accessibility]
**Priority:** [High / Medium / Low]

**Description:**
[What do you want and why?]

**Use Case:**
[How would you use this feature? What problem does it solve?]

**Additional Comments:**
[Mockups, references, or any other details]
```

## Tags

### Feature Type (user-applied)
- `Enhancement`
- `Modification`
- `Quality of Life`
- `Accessibility`

### Status (moderator-only)
- `Planned`
- `Under Review`
- `Implemented`
- `Won't Do`

## Field Descriptions

- **Type** — The kind of request. Use one of the available post tags.
  - **Enhancement:** A brand new feature that doesn't exist yet
  - **Modification:** A change to how an existing feature works
  - **Quality of Life:** A small UX improvement that makes things smoother
  - **Accessibility:** Improvements for usability and accessibility
- **Priority** — How important is this to you?
  - **High:** Significantly impacts your experience
  - **Medium:** Would be a nice improvement
  - **Low:** Minor wish, no rush

## Example Request

**Title:** Add queue reordering via drag and drop

**Type:** Enhancement
**Priority:** Medium

**Description:**
Allow users to reorder tracks in the play queue by dragging and dropping them. Currently the only option is to remove a track and re-add it in the desired position.

**Use Case:**
When listening to a playlist or album, I often want to move a specific track up next without losing the rest of my queue. Drag and drop would make this quick and intuitive.
